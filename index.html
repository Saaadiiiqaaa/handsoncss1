What is CSS?

  1. CSS Stands for Cascading Style sheet
  2. CSS describes how HTML Elements are to be displayed on paper,screen or in the other media
  3. CSS is not nessesary .CSS saves a lot of work. it can change the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS file.

CSS can be added in HTML documents in 3 ways

  • inline- by using style attribute inside html Elements
  • internal- by using style element inside head section
  • External- by using link element to link a external css file
  • the most common way to add css is to ,to keep the styles in external css file.however in this tutorial we will use inline and internal styles,because this is easier to demonstrate and easier to you to try it yourself

when comparing css class vs id ,the difference is that class file applies style to multiple Elements.ID,on the otherhand applies style to one unique element,ID is also special in that you can use a special url to link directly to an element which is used by java script.

in CSS, selectors are used to target specific element or range of elements on the webpage .once an element ha been targeted a style or set of style can be applied to the elements

there should be wide range of selector available .two of most commly used are class and ID.both are used to target elements to which style should be applied.
